2 Ziel des Projekts#

Generelles Ziel

Our goal is to build a foundation model for image segmentation. […]
We seek to develop a promptable model and pre-train it on a broad dataset using a task that enables powerful generalization. [KMR+23]

Foundation Model

Ein Foundation Model ist jedes Modell, das auf umfangreichen Daten trainiert wurde und an eine Vielzahl von weiterführenden Aufgaben angepasst bzw. fine-tuned werden kann.
Dabei wird im Allgmeinen self-supervised Training verwendet. Aktuelle Beispiele sind BERT, GPT-3 und CLIP. [BHA+22]

3 Anforderungen#

  • große Menge an Daten zum Trainieren

  • Fähigkeit von Zero-Shot Learning und Transfer

  • Unterstützung von flexiblen Prompts

  • Berechnung von Masken in Echtzeit

  • Mehrdeutigkeit von Objekten bewusst sein
